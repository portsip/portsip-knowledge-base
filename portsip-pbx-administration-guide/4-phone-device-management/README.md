# 4 Phone Device Management

With PortSIP PBX you can easily configure desk IP phones. Supported IP Phones such as Fanvil, Yealink, ALE, GrandStream, Snon, and Dinstar..., are tested and auto-configured with PortSIP PBX.  PortSIP PBX can provision these phones via:

* [Auto-provisioning security](auto-provisioning-security.md)
* [Custom IP Phone Template](custom-ip-phone-template.md)
* [Bulk Importing Users and Auto Provisioning IP Phones](bulk-importing-users-and-auto-provisioning-ip-phones.md)
* [Provision Phone Using PnP](provision-phone-using-pnp.md)
* [PnP Auto Provisioning IP Phone Multicast Debug](pnp-auto-provisioning-ip-phone-multicast-debug.md)
* [Provision Phone Using RPS](provision-phone-using-rps.md)
* [HTTP/HTTPS Provisioning link - Provisioned via the phone web interface. This is the method to use for legacy phones (e.g. Polycom, Cisco, or Aastra)](provision-phone-using-pnp.md#provision-phones-using-the-provisioning-link-manually)
* [Provision Phone Using DHCP Option 66](provision-phone-using-dhcp-option-66.md)
* [Provision Phone Using TFTP](provision-phone-using-tftp.md)
* [Provision Cisco 79xx IP Phones](provisioning-cisco-79xx-ip-phones.md)
