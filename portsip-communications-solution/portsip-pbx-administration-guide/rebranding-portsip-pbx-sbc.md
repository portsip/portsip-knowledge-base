# Rebranding PortSIP PBX, SBC

PortSIP PBX and PortSIP SBC include a simple but powerful **Rebranding** add-on that lets you customize the Web Portal (and related user-facing experiences) to match your company identity.

With this add-on, you can customize:

* Product name
* Company name
* Website link
* Logo
* Favicon
* Theme
* User-Agent string (PBX and SBC)
* WebRTC, Windows, and mobile apps
* IP phone templates
* Email notification templates

***

### Rebranding PortSIP PBX

#### Customize the User-Agent

You can change the **User-Agent** used by PortSIP PBX in SIP messages.

1. Sign in to the PortSIP PBX Web Portal using **System Admin** credentials.
2. Go to: **Advanced > Settings**
3. On the **General** page, locate the **User Agent** field.
4. Enter the User-Agent string you want to use, then save your changes.

***

#### Customize Company Branding (Name, Product, Links, Logo)

1. Sign in to the PortSIP PBX Web Portal using **System Admin** credentials.
2. Go to: **Advanced > Settings**
3. Open the **Rebranding** tab.
4. Update the following fields as needed:

* Web Portal Title
* Website Link
* Support Link
* Company Name
* Product Name
* Logo
* Favicon
* Theme
* Login page footer text (bottom text)

> ❗**Best Practice**\
> Upload a logo and favicon that match each theme’s color palette. This helps ensure your branding looks consistent across the Web Portal and the WebRTC client.

### Rebranding PortSIP SBC

#### Customize the User-Agent

You can change the **User-Agent** used by PortSIP SBC in SIP messages.

1. Sign in to the PortSIP SBC Web Portal.
2. Go to: **Settings > Advanced**
3. In the **User Agent** field, enter the User-Agent string you want to use, then save your changes.

***

#### Customize Company Branding (Name, Product, Links, Logo)

1. Sign in to the PortSIP SBC Web Portal.
2. Go to: **Settings > Rebranding**
3. Update the following fields as needed:

* Web Portal Title
* Website Link
* Support Link
* Company Name
* Product Name
* Logo
* Favicon
* Theme
* Login page footer text (bottom text)

> ❗**Best Practice**\
> Upload a logo and favicon that match each theme’s color palette. This provides a consistent branded experience in the Web Portal and the WebRTC client app.
