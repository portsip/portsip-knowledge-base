# 4 Phone Device Management

### Configuring Desk IP Phones

PortSIP PBX allows you to easily configure and provision desk IP phones.

Supported IP phones from [Fanvil](https://www.fanvil.com), [Yealink](https://www.yealink.com), [Grandstream](https://www.grandstream.com), [Snom](https://www.snom.com), [Gigaset](https://www.gigaset.com), and [Dinstar ](https://www.dinstar.com)have been fully tested with PortSIP PBX and support automatic provisioning.

***

* [Managing Phones](managing-phones.md)
* [Auto-Provisioning Security](auto-provisioning-security.md)
* [Custom IP Phone Templates](custom-ip-phone-template.md)
* [Bulk Import Users and Auto-Provision IP Phones](bulk-importing-users-and-auto-provisioning-ip-phones.md)

* [Provision Phones Using RPS](provision-phone-using-rps.md)
* [HTTP/HTTPS Provisioning Link (via the phone web interface; recommended for legacy phones such as Polycom, Cisco, and Aastra)](provision-phone-using-pnp.md#provisioning-phones-using-the-provisioning-link-manually)
* [Provision Phones Using DHCP Option 66](provision-phone-using-dhcp-option-66.md)
