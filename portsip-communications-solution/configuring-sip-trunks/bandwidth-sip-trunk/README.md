# Bandwidth SIP Trunk

[Bandwidth ](https://www.bandwidth.com/)is a communications platform as a service company. It sells software application programming interfaces for voice, text messaging, and emergency services, using its own global IP voice network.

PortSIP PBX natively integrates with Bandwidth SIP trunking to support both calling and SMS/MMS messaging. Within the PortSIP PBX configuration, users can select Bandwidth as their service provider, enabling seamless use of Bandwidthâ€™s voice and messaging services.

To successfully integrate Bandwidth with PortSIP PBX, you must complete the following high-level steps:

1. **Obtain a Bandwidth Account**\
   Create an account with Bandwidth to gain access to SIP trunking and messaging services.
2. **Log In and Configure Bandwidth Settings**\
   After account creation, sign in to your Bandwidth account and complete the required configurations for voice and SMS/MMS services.
3. **Configure Integration in PortSIP PBX**\
   Set up the Bandwidth SIP trunk and messaging integration in the PortSIP PBX Web Portal.
