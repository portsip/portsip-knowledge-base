# Purchase a DID on Voxtelesys Platform

Before placing or receiving calls, you must have at least one active DID (phone number) on your Voxtelesys account.\
If you already have one or more active DIDs, you may skip this section.

***

### Step 1: Access DID Ordering

1. Sign in to your [Voxtelesys ](https://portal.voxtelesys.net/)account.
2. Navigate to **Phone Numbers > Number Ordering**.

> ❗**Note**\
> You can also purchase DIDs by following the official Voxtelesys guide: [Order/Add DIDs.](https://voxtelesys.com/tutorial/voxtelesys-portal-order-add-dids)

***

### Step 2: Search for Available DIDs

Voxtelesys provides two search methods to help you find available numbers.

***

#### Option A: DID Search

Use **DID Search** to look up numbers based on specific geographic criteria.

Configure the following fields as needed:

* **Country** – Select **United States** or **Canada**
* **State** – Select a state or province
* **Rate Center** – Select a geographic rate center
* **Area Code / NPA** – Enter the desired area code
* **City Code / NXX** – Optional; narrows results to a specific exchange
* **Qty Displayed** – Choose how many numbers to display
* **Trunk Group** – Select an existing trunk group
  * Contact Voxtelesys Support if no trunk group is available
* **Consecutive** –
  * Enable to view consecutive numbers
  * Disable to view non-consecutive numbers

***

#### Option B: Bulk Search

Use **Bulk Search** to search across multiple area codes at once.

Configure the following fields:

* **Area Code / NPA** – Enter multiple area codes, one per line
* **Qty Displayed** – Select the number of results per area code (maximum 30)
* **Trunk Group** – Select an existing trunk group
* **Consecutive** –
  * Enable for consecutive numbers
  * Disable for non-consecutive numbers

<figure><img src="../../../.gitbook/assets/voxtelesys-fig2.png" alt=""><figcaption></figcaption></figure>

***

### Step 3: Select Phone Numbers

1. Select the DID number(s) you want to order.
2. Click **Next** to proceed to order review.

<figure><img src="../../../.gitbook/assets/voxtelesys-fig3.png" alt="" width="563"><figcaption></figcaption></figure>

***

#### Step 4: Review and Submit the Order

1. Review the **invoice details**.
2. Select a **payment method**.
3. Click **Submit Payment** to complete the order.

Once submitted, the DID numbers are assigned to your Voxtelesys account.
