# SIPTRUNK.COM SIP Trunk

[CM.com](https://cm.com) provides a robust, carrier-grade SIP Trunking solution that bridges your local or cloud PBX with the global telecommunications network. By combining high-quality voice connectivity with extensive messaging capabilities, we deliver a unified platform for both global calling and SMS services.

**PortSIP's** calling and SMS features allow users to select [CM.com](https://cm.com) as their service provider within the configuration settings, ensuring full utilization of both calling and messaging services. To successfully integrate [CM.com](https://cm.com) service with PortSIP PBX, users must follow these essential steps:

1. **Obtain a CM.com Account:** Begin by creating an account with [CM.com](https://cm.com) to access their services.
2. **Log In and Configure Settings:** After account creation, log in to your [CM.com](https://cm.com) account to set up the necessary configurations for seamless integration with PortSIP PBX.

PortSIP PBX integrated the [CM.com](https://cm.com) SIP trunking for calls and SMS/MMS, which allows you to set up easily.
